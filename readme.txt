在命令行下使用：

python AI2KD.py input.html
或
python AI2KD.py "input.html"

程序运行，生成了input_add_value.html和input_add_infl.html，input_add_infl.html就是最终添加了变形数据的文件。

注意，推荐第2种的方式，即使用引号包围输入文件名。当文件名中有空格时，请一定使用第2种方式。
